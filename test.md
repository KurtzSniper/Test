# Kinematics

$v = \frac{d}{t}$

$a = \frac{v-u}{t}$

$v = u + at$

$v^2 = u^2 + 2ad$

$d = \left(\frac{v+u}{2}\right)t$

$d = ut+\frac{1}{2}at^2$

$d = vt-\frac{1}{2}at^2$

# Forces

$F = ma$

$N = mg$ (on flat plane)

$f_{static} = \mu_{s} N$

$f_{kinetic} = \mu_{k} N$

$F_{y} = Fsin\theta$

$F_{x} = Fcos\theta$

$\Sigma F = F_{1}+F_{2}$

$\Sigma F = \sqrt{F_{1}^2+F_{2}^2 + 2F_{1}F_{2}cos\theta}$

$F = G\frac{m_{1} m_{2}}{r^2}$
